# Docker-Symfony

### Usage

Clone repository:

    $ git clone https://github.com/Maaiins/Docker-Symfony.git
    $ cd Docker-Symfony

Edit the `docker-compose.yml` for your needs

Try run:

    $ docker compose up --build
